Install and Use DATeS
======================

**DATeS can easily run without perminant installation:**

+ Prerequisits:
    
    - Python (both versions 2* and 3* are supported): https://www.python.org/            
    - Numpy: http://www.numpy.org/
    - Scipy: https://www.scipy.org/
    - Scikit-learn: http://scikit-learn.org/stable/
    - Matplotlib: https://matplotlib.org/
    
    - gfortran compiler: https://gcc.gnu.org/wiki/GFortran
    
    *Note:*
        - To install the scipy stack, visit: https://www.scipy.org/install.html
        
        - To install scikitlearn, visit: http://scikit-learn.org/stable/install.html
